# 1D Convolutional Neural Networks(1D CNN)

순차적 데이터에 적용되는 합성곱 신경망으로, 특히 자연어 처리, 시계열 분석, 음성 신호 처리와 같은 1차원 형태의 데이터를 처리하는 데 효과적입니다.

## 1D CNN의 구조와 동작 원리

1D CNN은 입력 데이터에 대해 1차원 **필터(kernel)**를 적용하여 특정 패턴을 감지합니다. 

이 필터는 데이터의 시간적 순서나 문맥적 패턴을 학습하는 데 유용합니다.

### 주요 구성 요소

**입력 데이터** 

    입력 데이터는 보통 1차원 벡터의 형태로 주어지며, 각 벡터의 요소가 시간 또는 위치 순서에 따라 배열된 값들입니다.

    예를 들어, 문장의 단어 임베딩 벡터나 시계열 데이터가 이에 해당합니다.

**1D 합성곱(Convolution) 연산**

    1D CNN은 필터를 데이터의 한 축(시간 또는 위치) 방향으로만 이동시키며 합성곱 연산을 수행합니다.

    이 필터는 입력 데이터의 특정 패턴을 감지하며, 필터가 데이터에서 유용한 특징을 찾아내도록 학습됩니다.

**필터와 패딩(Padding)**

    필터의 크기와 패딩 옵션에 따라 출력의 크기가 달라집니다. 

    일반적으로 valid padding(출력 크기가 입력보다 작음)이나 same padding(출력 크기가 입력과 같음)을 설정하여 데이터의 가장자리를 처리할 수 있습니다.

**활성화 함수(Activation Function)**

    각 합성곱 연산 후 활성화 함수를 적용하여 비선형성을 추가하고, 모델이 복잡한 패턴을 학습할 수 있도록 합니다. 

    주로 ReLU가 사용됩니다.


**풀링(Pooling) 레이어**

    필요에 따라 1D MaxPooling을 적용하여 특성의 크기를 줄이고 모델의 학습 속도를 높이며, 과적합을 방지합니다.


### 1D CNN의 구조

- 입력: 길이가 100인 1D 시퀀스
- 합성곱 레이어: 1D 필터 크기 3, 필터 개수 64, 활성화 함수 ReLU
- 풀링 레이어: 1D MaxPooling, 크기 2
- 합성곱 레이어: 1D 필터 크기 3, 필터 개수 128, 활성화 함수 ReLU
- 풀링 레이어: 1D MaxPooling, 크기 2
- 밀집 레이어(Dense): 출력 크기 1 (예: 이진 분류의 경우)

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv1D, MaxPooling1D, Flatten, Dense

model = Sequential([
    Conv1D(64, 3, activation='relu', input_shape=(100, 1)),
    MaxPooling1D(2),
    Conv1D(128, 3, activation='relu'),
    MaxPooling1D(2),
    Flatten(),
    Dense(1, activation='sigmoid')
])
```

### 1D CNN의 장점

효율적인 패턴 인식: 1D CNN은 데이터의 순차적 특성을 고려하며, 특정 위치에 구애받지 않고 패턴을 감지할 수 있어, 문장의 의미적 유사성을 파악하거나 시계열 패턴을 분석하는 데 유리합니다.

파라미터 절감: 1D CNN은 RNN에 비해 상대적으로 적은 파라미터를 사용하며, 순차 데이터에서 긴 의존성을 효과적으로 학습할 수 있습니다.

병렬 처리 가능: 1D CNN은 전체 시퀀스를 한 번에 처리할 수 있어 RNN과 비교했을 때 병렬 처리 및 연산 속도 측면에서 유리합니다.

## 1D CNN의 활용 사례

자연어 처리(NLP): 문장 내 특정 n-gram 패턴을 학습하여 텍스트 분류, 감정 분석 등에 사용됩니다.

시계열 데이터 분석: 주식 가격, 날씨 패턴, 센서 데이터의 변동을 분석하는 데 활용됩니다.

의료 데이터 처리: 심전도(ECG), 뇌파(EEG)와 같은 생체 신호 분석에도 사용됩니다.
