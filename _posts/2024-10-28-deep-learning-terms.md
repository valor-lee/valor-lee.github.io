# 개요

인공 신경망(Artificial Neural Network)의 층을 연속적으로 깊게 쌓아올려 데이터를 학습하는 방식을 말합니다.

# 딥러닝 용어

## 퍼셉트론(Perceptron)

초기 형태의 인공 신경망으로, 다수의 입력으로부터 하나의 결과를 내보내는 알고리즘입니다.

## 활성화 함수(Activation Function)

출력값을 변경시키는 함수를 의미합니다.

계단 함수, 시그모이드 함수, **하이퍼볼릭탄젠트 함수**, 소프트맥스 함수, **렐루 함수(ReLU)**, **리키 렐루(Leaky ReLU)** 등이 있습니다.

활성화 함수는 비선형 함수여야 합니다.
- 층을 깊게 만들지 못하기 때문입니다.




## 단층 퍼셉트론(Single-Layer Perceptron)

입력층과 출력층 두 단계로만 이루어지는 구조를 말합니다.

단층 퍼셉트론은 직선 하나로 두 영역을 나눌 수 있는 문제만 구현할 수 있기 때문에 **XOR 게이트는 구현할 수 없습니다.**


## 다층 퍼셉트론(MultiLayer Perceptron, MLP)

입력층과 출력층 두 단계로만 사이에 은닉층을 추가한 구조를 말합니다.

 은닉층이 2개 이상인 신경망을 **심층 신경망(Deep Neural Network, DNN)** 이라고 합니다.

## 피드 포워드 신경망(Feed-Forward Neural Network, FFNN)

오직 입력층에서 출력층 방향으로 연산이 전개되는 신경망을 뜻합니다.


## 전결합층(Fully-connected layer, FC, Dense layer)

어떤 층의 모든 뉴런이 이전 층의 모든 뉴런과 연결돼 있는 층을 의미합니다.



## 순전파(Foward Propagation)

입력층에서 출력층 방향으로 예측값의 연산이 진행되는 과정을 의미합니다.

임시의 가중치와 편향을 기반으로 예측치를 계산하는 과정입니다.

## 손실 함수(Loss Function)

실제값과 예측값의 차이를 수치화해주는 함수입니다.

회귀에서는 평균 제곱 오차, 분류 문제에서는 크로스 엔트로피를 주로 손실 함수로 사용합니다.

**MSE(Mean Squared Error, MSE)** 
-  연속형 변수를 예측할 때 사용됩니다.

**이진 크로스 엔트로피(Binary Cross-Entropy)**
- 이항 교차 엔트로피라고도 부르는 손실 함수입니다.
- 출력층에서 시그모이드 함수를 사용하는 이진 분류 (Binary Classification)의 경우 binary_crossentropy를 사용합니다. 

**카테고리칼 크로스 엔트로피(Categorical Cross-Entropy)**
- 범주형 교차 엔트로피라고도 부르는 손실 함수입니다.
-  출력층에서 소프트맥스 함수를 사용하는 다중 클래스 분류(Multi-Class Classification)일 경우 categorical_crossentropy를 사용합니다.
- 레이블에 대해서 원-핫 인코딩 과정을 생략하고, 정수값을 가진 레이블에 대해서 다중 클래스 분류를 수행하고 싶다면 다음과 같이 'sparse_categorical_crossentropy'를 사용합니다.

[그 외 손실함수들](https://www.tensorflow.org/api_docs/python/tf/keras/losses)


## 배치 크기에 따른 경사 하강법

매개 변수의 값을 조정하기 위해 사용하는 데이터의 양, 배치(Batch)의 크기에 따라 방법론을 나눌 수 있습니다.

배치 경사 하강법(Batch Gradient Descent)
- 한 번의 에포크에 모든 매개변수 업데이트를 단 한 번 수행합니다.
- 전체 데이터를 고려해서 학습하므로 한 번의 매개 변수 업데이트에 시간이 오래 걸리며, 메모리를 크게 요구한다는 단점이 있습니다.

확률적 경사 하강법(Stochastic Gradient Descent, SGD)
- 매개변수 값을 조정 시 전체 데이터가 아니라 랜덤으로 선택한 하나의 데이터에 대해서만 계산하는 방법입니다.

미니 배치 경사 하강법(Mini-Batch Gradient Descent)
- 배치 크기를 지정하여 해당 데이터 개수만큼에 대해서 계산하여 매개 변수의 값을 조정하는 방법입니다.
- 일반적으로 2의 n제곱에 해당하는 숫자로 선택하는 것l이 보편적입니다.

## optimizer

손실함수 값을 줄여나가면서 파라미터를 업데이트하는 역전파 과정을 수행합니다.

모멘텀(Momentum)
- 경사 하강법에서 계산된 접선의 기울기에 한 시점 전의 접선의 기울기값을 일정한 비율만큼 반영합니다.
- 현재의 로컬 미니멈에서 탈출하고 글로벌 미니멈 내지는 더 낮은 로컬 미니멈으로 갈 수 있는 효과를 얻을 수도 있습니다.

아다그라드(Adagrad)
- 변화가 많은 매개변수는 학습률이 작게 설정되고 변화가 적은 매개변수는 학습률을 높게 설정시켜서 매개변수별 학습률을 차등하는 방식입니다.

알엠에스프롭(RMSprop)
- 아다그라드의 반복된 학습 시 학습률이 떨어지는 단점을 보완한 방식입니다.

아담(Adam)
- 알엠에스프롭과 모멘텀 두 가지를 합친 듯한 방법으로, 방향과 학습률 두 가지를 모두 잡기 위한 방법입니다.

[옵티마이저 사용법](https://www.tensorflow.org/api_docs/python/tf/keras/optimizers)

## 에포크와 배치크기 이터레이션

**에포크**
- 인공 신경망에서 전체 데이터에 대해서 순전파와 역전파가 끝난 상태를 말합니다.

**배치 크기**
- 몇 개의 데이터 단위로 매개변수를 업데이트 하는지를 말합니다.
- 전체 데이터가 2,000일때 배치 크기를 200으로 준다면 배치의 수는 10입니다.
- 이때 배치의 수를 이터레이션이라고 합니다.
  - 즉, 한 번의 에포크를 끝내기 위해서 필요한 배치의 수이며, 스텝(step)이라고도 부릅니다.


## 역전파(Backward Propagation)

출력층에서 입력층 방향으로 계산하면서 가중치를 업데이트해갑니다.

출력층 바로 이전의 은닉층을 N층이라고 하였을 때, 출력층과 N층 사이의 가중치를 업데이트하는 단계를 역전파 1단계, 

그리고 N층과 N층의 이전층 사이의 가중치를 업데이트 하는 단계를 역전파 2단계라고 합니다.

### 업데이트 방식

![전체오차 미분의 연쇄법칙](https://github.com/user-attachments/assets/1025ebd9-d36f-4d2e-a652-dd541f01de17)

i 번째 가중치에 대한 전체 오차의 미분의 연쇄 법칙을 이용합니다.

- *E<sub>total</sub>* : 순전파를 계산했던 전체 오차값
- *O<sub>i</sub>* : 시그모이드 함수의 출력값
    - [시그모이드 함수 미분 참고 링크](https://en.wikipedia.org/wiki/Logistic_function#Derivative)
- 세번째 항은 *b<sub>i</sub>* 와 동일합니다.

위 세 개의 값을 곱과 학습률을 곱해서 기존 가중치에서 빼면 됩니다.

역전파 i 번째도 이와 같이 연쇄법칙을 이용하여 값을 구해나가면 됩니다.




# 과적합 방지법

## 데이터 양 늘리기

데이터 양을 늘려서 모델이 데이터의 일반적인 페턴을 학습하도록 할 수 있습니다.

데이터 양이 적을 경우 데이터 증식 또는 증강을 통해 데이터 양을 늘리기도 합니다.

- 텍스트의 경우 번역 후 재번역을 통해 새로운 데이터를 만들어 내는 역번역 등의 방법이 있습니다.

## 모델의 복잡도 줄이기

인공신경망의 복잡도를 줄이는, 즉 매개변수들의 수를 줄이는 것이 도움이 됩니다.

## 가중치 규제(Regularization)

복잡한 모델을 좀 더 간단하게 하는 방법입니다.

규제 강도를 정하는 하이퍼파라미터를 곱해주면서 규제 정도를 조절할 수 있습니다.
- 하이퍼파라미터 값이 크다면 모델이 훈련 데이터에 대해 적합한 매개변수를 찾는 것보다 규제를 위해 추가된 항들을 작게 유지하는 것을 우선으로 한다는 뜻입니다.

### L1 규제

가중치 w들의 절대값 합계를 비용 함수에 추가합니다. L1 노름이라고도 합니다.

L1 규제를 사용하면 비용 함수가 최소가 되게 하는 가중치와 편향을 찾는 동시에

**가중치들의 절대값의 합도 최소가 되어야 합니다.**

이로 인해, 가중치 w의 값들은 0 또는 0에 가까이 작아져야 하므로 어떤 특성들은 모델을 만들 때 거의 사용되지 않게 됩니다.

그래서 어떤 특성들이 모델에 영향을 주고 있는지를 정확히 판단하고자 할 때 유용합니다.

### L2 규제

모든 가중치 w들의 제곱합을 비용 함수에 추가합니다. L2 노름이라고도 합니다.
- 인공신경망에서는 가중치 감쇠(weight decay)라고도 부릅니다.

L1 규제와는 달리 가중치들의 제곱을 최소화하므로 w의 값이 완전히 0이 되기보다는 0에 가까워지기는 경향을 띕니다.

특성들의 모델에 대한 영향도를 알 필요가 없다면, L2 규제가 더 잘 동작하므로 L2 규제를 더 권장합니다.


## 드롭아웃(Dropout)

학습과정에서 일부 신경망을 사용하지 않는 방법입니다.

드롭아웃의 비율을 0.5로 한다면 학습 과정마다 랜덤으로 절반의 뉴런을 사용하지 않고, 절반의 뉴런만을 사용합니다.

드롭아웃은 신경망 학습 시에만 사용하고, 예측 시에는 사용하지 않는 것이 일반적입니다.

학습 시에 인공 신경망이 특정 뉴런 또는 특정 조합에 너무 의존적이게 되는 것을 방지해줍니다.

또한, 매번 랜덤 선택으로 뉴런들을 사용하지 않으므로 서로 다른 신경망들을 앙상블하여 사용하는 것 같은 효과를 내어 과적합을 방지합니다.

#  기울기 소실(Gradient Vanishing)과 폭주(Exploding)

순환 신경망(Recurrent Neural Network, RNN)에서 쉽게 발생할 수 있는 현상입니다.

기울기 소실
- 역전파 과정에서 입력층으로 갈 수록 기울기(Gradient)가 점차적으로 작아지는 현상입니다.
- 깊은 인공 신경망에서 발생하는 현상입니다.

기울기 폭주
- 기울기가 점차 커지더니 가중치들이 비정상적으로 큰 값이 되면서 결국 발산되기도 합니다.

## ReLU와 ReLU의 변형들

은닉층의 활성화 함수로 시그모이드나 하이퍼볼릭탄젠트 함수 대신에 ReLU나 ReLU의 변형 함수와 같은 Leaky ReLU를 사용하는 것입니다.

## 그래디언트 클리핑(Gradient Clipping)

기울기 폭주를 막기 위해 임계값을 넘지 않도록 값을 자릅니다.

RNN에서 유용합니다.
- RNN은 역전파 과정에서 시점을 역행하면서 기울기를 구하는데, 이때 기울기가 너무 커질 수 있기 때문입니다.

## 가중치 초기화(Weight initialization)

가중치가 초기에 어떤 값을 가졌느냐에 따라서 모델의 훈련 결과가 달라지기도 합니다.

### 세이비어 초기화(Xavier Initialization)

이전 층 뉴런 개수와 다음 층 뉴런 개수를 고려하여 균등 분포(Uniform Distribution) 또는 정규 분포(Normal distribution)를 사용한 초기화 방법론입니다.

여러 층의 기울기 분산 사이에 균형을 맞춰서 특정 층이 너무 주목을 받거나 다른 층이 뒤쳐지는 것을 막습니다.

시그모이드 함수나 하이퍼볼릭 탄젠트 함수와 같은 S자 형태인 활성화 함수와 함께 사용할 경우에는 좋은 성능을 보이지만,

ReLU 함수 또는 ReLU의 변형 함수들을 활성화 함수로 사용할 경우에는 성능이 좋지 않습니다.


[관련 논문](http://proceedings.mlr.press/v9/glorot10a/glorot10a.pdf)


## He 초기화(He initializtion)

이전 층의 뉴런의 개수에 따라 정규 분포 또는 균등 분포로 초기화하는 방법입니다.

ReLU 계열 함수를 사용할 경우에는 He 초기화 방법이 효율적입니다.

ReLU + He 초기화 방법이 좀 더 보편적입니다.

[관련 논문](https://www.cv-foundation.org/openaccess/content_iccv_2015/papers/He_Delving_Deep_into_ICCV_2015_paper.pdf)


## 배치 정규화(Batch Normalization)

인공 신경망의 **각 층에 들어가는 입력을 평균과 분산으로 정규화**하여 학습을 효율적으로 만듭니다.

### 내부 공변량 변화(Internal Covariate Shift)

학습 과정에서 층 별로 입력 데이터 분포가 달라지는 현상을 말합니다. 

이전 층들의 학습에 의해 이전 층의 가중치 값이 바뀌게 되면, 현재 층에 전달되는 입력 데이터의 분포가 현재 층이 학습했던 시점의 분포와 차이가 발생합니다.

- **공변량 변화**는 훈련 데이터의 분포와 테스트 데이터의 분포가 다른 경우를 의미합니다.
- **내부 공변량 변화**는 신경망 층 사이에서 발생하는 입력 데이터의 분포 변화를 의미합니다.


### 배치 정규화

학습 시 배치 단위의 평균과 분산들을 차례대로 받아 이동 평균과 이동 분산을 저장해놓았다가 

테스트 할 때는 해당 배치의 평균과 분산을 구하지 않고 구해놓았던 평균과 분산으로 정규화를 합니다.

- 배치 정규화를 사용하면 시그모이드 함수나 하이퍼볼릭탄젠트 함수를 사용하더라도 기울기 소실 문제가 크게 개선됩니다.
- 가중치 초기화에 훨씬 덜 민감해집니다.
- 훨씬 큰 학습률을 사용할 수 있어 학습 속도를 개선시킵니다.

Subclassing API는 밑바닥부터 새로운 수준의 아키텍처를 구현해야 하는 실험적 연구를 하는 연구자들에게 적합합니다.

## 세 가지 구현 방식 비교


1. Sequential API

장점 : 단순하게 층을 쌓는 방식으로 쉽고 사용하기가 간단합니다.
단점 : 다수의 입력(multi-input), 다수의 출력(multi-output)을 가진 모델 또는 층 간의 연결(concatenate)이나 덧셈(Add)과 같은 연산을 하는 모델을 구현하기에는 적합하지 않습니다.

2. Functional API

장점 : Sequential API로는 구현하기 어려운 복잡한 모델들을 구현할 수 있습니다.
단점 : 입력의 크기(shape)를 명시한 입력층(Input layer)을 모델의 앞단에 정의해주어야 합니다.

3.  Subclassing API
장점 : Functional API로도 구현할 수 없는 모델들조차 구현이 가능합니다.
단점 : 객체 지향 프로그래밍(Object-oriented programming)에 익숙해야 하므로 코드 사용이 가장 까다롭습니다.

# 피드 포워드 신경망 언어 모델(Feed Forward Neural Network Language Model, FFNNLM)

신경망 언어 모델의 시초입니다.

이전 통계 기반 언어 모델은 충분한 데이터를 학습하지 못하면 언어를 정확히 모델링하지 못하는 희소 문제가 있었습니다.

즉, 기계가 단어의 의미적 유사성을 알지 못한다는 문제입니다.

이를 해결하기 위해 단어 벡터간 유사도를 구할 수 있는 벡터를 얻어내어(워드 임베딩, Word Embedding) 인공신경망을 학습시키는 방범론이 제시되었습니다.

피드 포워드 신경망 언어 모델(Neural Network Language Model, NNLM)은 신경망을 이용해 언어의 확률 분포를 학습하는 초기 언어 모델입니다. 

NNLM은 문장의 다음 단어가 나올 확률을 예측하는 데 주로 사용되며, 

고정된 개수의 단어를 입력으로 받아 **미리 정의된 고정 길이의 문맥(window)**을 기반으로 다음 단어를 예측합니다.

## NNLM의 주요 특징

1. 피드포워드 구조: NNLM은 단순한 다층 퍼셉트론(MLP) 구조를 사용해 입력 단어의 임베딩 벡터를 학습하고 이를 연결하여 예측을 수행합니다.
2. 고정된 윈도우 크기: 일반적으로 n개의 단어(고정된 문맥 창 크기)를 사용해 다음 단어를 예측합니다. 문맥 창 크기 이상으로 먼 단어는 학습에 반영되지 않는다는 단점이 있습니다.
3. 단어 임베딩 학습: 입력 단어를 벡터화하여 희소한 단어 표현을 밀집 벡터(dense vector)로 변환하며, 이 과정에서 단어 간 의미적 유사성이 학습됩니다.
4. 모델의 구성: 임베딩 레이어 -> 은닉 레이어 -> 소프트맥스 레이어로 구성되며, 소프트맥스 레이어를 통해 다음 단어의 확률 분포를 계산합니다.


## NNLM의 구성요소


1. 임베딩 레이어(Embedding Layer)

    입력으로 주어진 단어들을 벡터로 변환하는 레이어입니다.
    
    모델은 n개의 단어를 고정된 크기의 임베딩 벡터로 변환합니다.
    
    여기서 각 단어는 고유한 벡터(밀집 표현)로 매핑되며, 학습 과정을 통해 벡터들이 단어 간의 의미적 유사성을 반영하도록 최적화됩니다.
    
    예를 들어, 단어 수(n)가 10000개이고 임베딩 차원(m)이 300이라면, 이 임베딩 레이어는 10000 x 300의 임베딩 행렬로 구성됩니다.

    입력된 n개의 임베딩 벡터를 하나의 벡터로 결합(Concatenate)하여, n * m 차원의 백터로 합쳐집니다. 

2. 은닉층(Hidden Layer)

    합쳐진 벡터로 은닉층을 통해 다음 단어의 조건부 확률을 예측합니다.
    
    보통 ReLU나 Tanh 같은 활성화 함수가 적용되며, 이 과정을 통해 모델이 문맥에 맞는 다음 단어를 예측할 수 있도록 문맥 정보를 학습합니다.
    
    NNLM에서는 입력이 고정된 n개의 단어이기 때문에, 피드포워드 신경망으로 은닉층을 구성하여 다음 단어를 계산하는 구조가 됩니다.

3. 출력층(Softmax Layer)

    은닉층의 출력을 소프트맥스 함수를 통해 다음 단어의 확률 분포로 변환하는 레이어입니다.

    소프트맥스는 전체 단어 집합에 대해 각 단어의 조건부 확률을 계산하여 가장 확률이 높은 단어를 예측하도록 합니다.
    
    예를 들어, 단어 집합의 크기가 10000이라면, 출력층은 10000차원의 확률 벡터를 생성하여 다음 단어의 확률을 반환합니다.


## NNLM의 동작 흐름 예시

단계 1: 모델에 입력된 n개의 단어를 임베딩 레이어에서 밀집 벡터로 변환.

단계 2: 변환된 임베딩 벡터를 연결하여 은닉층에 전달하고, 활성화 함수를 통해 다음 단어의 문맥 정보를 학습.

단계 3: 은닉층의 출력을 소프트맥스 레이어에 입력해 전체 단어 집합의 각 단어에 대한 확률 분포를 생성.

단계 4: 확률 분포에서 가장 높은 확률을 갖는 단어를 다음 단어로 예측.

이와 같은 구성으로 NNLM은 고정된 크기의 문맥 내에서 다음 단어의 확률을 예측하며, 단어 임베딩 학습을 통해 유의미한 단어 간 관계를 학습할 수 있습니다.


## NNLM의 이점과 한계

기존 모델인 n-gram 모델에서 단어간 유사도를 판단할 수 없다는 문제를 임베딩 벡터를 통해 개선하며 희소 문제를 해결하였습니다.

하지만 여전히 고정된 n 개의 단어만 참고할 수 있다는 단점이 존재하고 있습니다.

이를 보완하기 위해 RNN(Recurrent Neutral Network)를 사용한 RNN언어모델(RNNLM)이 사용할 수 있습니다.
